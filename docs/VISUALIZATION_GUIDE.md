## ğŸ“ˆ Using Visualizations

### Generate All Visualizations

```bash
# Generate comprehensive summary
python scripts/generate_visualization_summary.py
```

This creates:
- All plot images in `outputs/plots/`
- HTML summary report in `outputs/reports/`

### View Interactive Dashboard

```bash
# Start results dashboard
python run_dashboard.py results

# Or use module directly
python -m lca_optimizer.visualization.results_dashboard run_results_dashboard
```

### Programmatic Usage

```python
from lca_optimizer.visualization.plots import (
    plot_lca_results, plot_policy_impact, plot_time_series_lca
)

# Plot LCA breakdown
plot_lca_results(
    breakdown={"upstream": 100, "process": 200, "downstream": 50},
    title="Steel LCA Breakdown",
    save_path="outputs/plots/steel_breakdown.png"
)

# Plot policy impact
plot_policy_impact(
    baseline=1800000.0,
    impacts={
        "Policy 1": {"new_emissions": 1200000, "reduction_percentage": 33.3}
    },
    save_path="outputs/plots/policy_impact.png"
)
```
